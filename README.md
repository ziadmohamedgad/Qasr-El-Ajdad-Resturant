# Qasr El-Ajdad Restaurant Landing Page

Welcome to the official landing page project for Qasr El-Ajdad Restaurant. This page is designed to be an attractive and modern digital storefront that reflects the restaurant's authentic identity while providing a seamless user experience for ordering food and getting in touch.

## 🚀 Project Overview

A fully responsive, single-page website tailored for a restaurant serving authentic Arabic cuisine, such as Mandi and Madhbi. The page forces a Light Mode theme to strictly preserve the visual brand identity and functions perfectly across all devices (mobile, tablet, desktop).

## ✨ Key Features

* **Modern & Attractive Design:** A UI that utilizes dark brown and gold color palettes to reflect premium quality and heritage.
* **100% Responsive:** Layout works flawlessly on mobile phones and all screen sizes.
* **Interactive Meals Carousel:** A smooth, touch-friendly (swipeable) image slider showcasing top dishes. Fully optimized to work correctly with Right-to-Left (RTL) text direction.
* **Delivery App Integration:** Direct order cards for popular delivery platforms (HungerStation, Jahez, Keeta) featuring scannable QR codes and actual brand logos.
* **Animations (Hover & Scroll):** Engaging scale effects on cards when hovered, along with smooth scroll-reveal animations for elements as they enter the viewport.
* **Color Scheme Lock:** Prevents mobile browsers from forcefully applying "Dark Mode" color inversions, keeping the brand's colors accurate at all times.
* **Direct Contact Links:** One-click buttons for direct phone calls and WhatsApp messaging.
* **Map Integration:** Direct link to the exact restaurant location on Google Maps.

## 🛠 Technologies Used

* **HTML5:** Semantic markup structure optimized for SEO.
* **CSS3 (Vanilla):** Styling, responsiveness (`@media queries`), CSS Variables for easy theming, animations, and typography using the "Tajawal" Google Font.
* **JavaScript (Vanilla JS):** Handles frontend logic including the mobile hamburger menu, sticky navbar scrolling, scrollspy (active link highlighting), and the custom image carousel.
* **Font Awesome:** Icon library used throughout the page.

## ⚙️ Setup & Usage

This project does not require any special build tools, dependencies, or a backend server.
To view the site:
1. Extract or clone the project files.
2. Open `index.html` directly in any modern web browser (Google Chrome, Firefox, Safari, Edge).

## 📄 Attached Documentation

2. **Technical_Documentation.html:** Explains backend/tech details for developers.

## 👨‍💻 Development & Editing

* To edit text content, phone numbers, or links: Modify `index.html`.
* To edit the primary brand colors or spacing: Modify the CSS variables inside the `:root` pseudo-class at the top of `style.css`.
